Skills
Building a website is an art, and there are many tools available to web developers right now to create them. Here are a couple of skills and tools I emply in projects:

HTML, CSS, Javascript

HTML, CSS, and Javascript are a staple of any web developers toolkit. A majority of my experience and time has been spent creating and cultivating static website creation. Aside from the basic languages, I now work with the SASS preprocessor and the Jquery library to help increase programming efficiency and website capacities.

Angular 2

To diversify my skills and to allow for more dynamic and robust website building, I have employed the use of Angular 2 in my web applications. Angular 2 is my first choice in a front end framework for its delivery of an entire website framework and the use of Typescript as its main language.

Node.Js

For all of my backend coding, I use node.js. Node allows for the use of one language throughout the entire stack, javascript, which has become my most enjoyed and used language in my kit. Along with the use of Express, Mongoose, and NPM, I have created applications with fully functioning databases and dynamic functionality.
